# PlexTracker ⚡
**The Ultimate Zero-Equipment Fitness & Habit Dashboard**

PlexTracker is a frontend web application designed to track body recomposition, zero-equipment calisthenics protocols, and elite daily habits. Built with a focus on speed, aesthetics, and data visualization, it replaces bloated fitness apps with a seamless, zero-friction daily dashboard.

---

## 🚀 Features

* **Zero-Equipment Protocol:** Pre-built 6-day calisthenics workout splits (Push, Pull, Legs, Core, HIIT, Recovery).
* **Dynamic Data Visualization:** Integrates Chart.js to render real-time weekly volume bar charts and training split donut charts.
* **Automated Midnight Reset:** JavaScript engine detects new calendar days to automatically wipe daily tasks and water intake while preserving long-term weekly data.
* **4L Hydration Tracking:** Interactive visual water tracking system.
* **Local Storage Engine:** 100% offline functionality. All user data is securely stored in the browser's local memory.
* **Responsive Dark UI:** Premium, high-contrast neon aesthetic optimized for both desktop and mobile viewing.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and custom SVG vector graphics.
* **CSS3:** Custom CSS variables, CSS Grid/Flexbox layouts, and fully responsive media queries.
* **JavaScript (ES6):** DOM manipulation, date-checking algorithms, and Local Storage data management.
* **Chart.js:** Third-party library for rendering dynamic, data-driven HTML5 canvas charts.

---

## 💻 How to Run Locally

Because PlexTracker relies purely on frontend web technologies and local storage, no backend server is required for version 1.0.

1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Safari, Edge).
3. Click "Launch Dashboard" to navigate to `dashboard.html` and begin tracking.
